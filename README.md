# เจ้น้อย เขียงหมูตะคร้อ

เว็บไซต์สั่งซื้อสินค้าสำหรับร้านเจ้น้อย เขียงหมูตะคร้อ ออกแบบแบบ mobile-first และใช้ภาษาไทยเป็นหลัก

## ฟังก์ชันหลัก

- เลือกสินค้าและปรับจำนวนในตะกร้า
- กรอกข้อมูลผู้รับและที่อยู่จัดส่ง
- แนบสลิปการชำระเงิน
- บันทึกออเดอร์และไฟล์แบบถาวร
- หลังบ้านสำหรับตรวจสลิปและเปลี่ยนสถานะออเดอร์

## ข้อมูลที่รอยืนยัน

- ราคาไส้กรอกอีสานแพ็กละ 10 ชิ้น
- น้ำหนักแคปหมูต่อกล่อง
- ค่าจัดส่ง
- เลขพร้อมเพย์และชื่อบัญชี

## เริ่มพัฒนา

```bash
npm install
npm run dev
```

ตรวจความพร้อมก่อนเผยแพร่ด้วย `npm run lint` และ `npm run build`
