# 🛠️ คู่มือการใช้งาน Figma Plugin (วาดแผนภาพ UX/UI เจ๊น้อย)

นี่คือปลั๊กอิน Figma ท้องถิ่น (Local Figma Plugin) ที่สร้างขึ้นมาเพื่อช่วยดึงภาพหน้าจอจริงทั้ง 14 หน้าจากการใช้งานเว็บบนเครื่องของคุณ นำไปเรียงลำดับวาดเป็นแผนภาพ Flowchart ที่สวยงาม จัดวางสัดส่วน พิกัด และวาดลูกศรนำทางสีแดงตามแบรนด์เจ๊น้อยให้อัตโนมัติบนบอร์ด Figma ของคุณ

---

## 🚀 ขั้นตอนการติดตั้งและเรียกใช้งานปลั๊กอิน (ใน 3 ขั้นตอน)

### ขั้นตอนที่ 1: ลงทะเบียนปลั๊กอินในแอปพลิเคชัน Figma
1. เปิดโปรแกรม **Figma Desktop App** หรือเปิดบนเว็บเบราว์เซอร์แล้วเข้าไปที่โปรเจกต์ของคุณ
2. คลิกปุ่ม **Resources** (ไอคอนรูปสี่เหลี่ยม/เลเยอร์ด้านซ้ายบน หรือกดปุ่มลัด `Shift + I`) 
3. เลือกแท็บ **Plugins** จากนั้นกดแท็บย่อย **Development**
4. กดเครื่องหมายบวก **`+`** (หรือคลิกขวาเลือก *Plugins* -> *Development* -> *New plugin...*)
5. ในหน้าต่างที่เด้งขึ้นมา ให้เลือกหัวข้อ **Import plugin from manifest...** (อยู่ด้านล่างสุด)
6. ค้นหาโฟลเดอร์ปลั๊กอินนี้ในเครื่องของคุณ แล้วเลือกไฟล์ **[manifest.json](file:///Users/earthondev/Desktop/jae-noi-pork-shop/figma-plugin/manifest.json)** เพื่อลงทะเบียนเข้าระบบ

---

### ขั้นตอนที่ 2: เปิดใช้งานปลั๊กอิน
1. เมื่อลงทะเบียนสำเร็จแล้ว จะพบรายชื่อปลั๊กอิน **"Jae Noi UX Flowchart Generator"** แสดงขึ้นมาในรายการ Development Plugins
2. กดปุ่ม **Run** เพื่อเปิดแผงควบคุมการอัปโหลด

---

### ขั้นตอนที่ 3: เลือกอิมพอร์ตภาพหน้าจอ
1. แผงควบคุมจะแสดงช่องสำหรับ Drag & Drop
2. เปิดหน้าต่างจัดการไฟล์ (Finder ใน macOS) แล้วเข้าไปที่โฟลเดอร์ภาพหน้าจอ:
   📂 **[storefront-ux-flow](file:///Users/earthondev/Desktop/jae-noi-pork-shop/output/playwright/storefront-ux-flow/)**
3. ลากคลุมไฟล์ภาพทั้ง 14 ไฟล์ (ชื่อไฟล์ `01-*.png` ถึง `14-*.png`) แล้วทำการ **ลากมาวาง (Drag & Drop)** ลงในหน้าต่างปลั๊กอิน Figma
4. ตรวจสอบลำดับไฟล์ว่าเรียงลำดับ 1 ถึง 14 ถูกต้อง จากนั้นกดปุ่ม **"วาดแผนภาพลง Figma Canvas"**
5. ปลั๊กอินจะทำการประมวลผล วาดภาพ ปรับแต่งขนาด Desktop/Mobile ใส่เงา (Drop Shadow) เขียนหัวข้อภาษาไทย และวาดลูกศรเชื่อมแต่ละหน้าจอให้คุณทันทีแบบเรียลไทม์!
