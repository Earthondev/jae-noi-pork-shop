# Thailand address data source

- Source: GeoThai `geothai` npm package
- Source repository: https://github.com/GeoThai/npm-packages
- Package version: 3.0.1
- License: MIT
- Generated fields: Thai province, district, subdistrict, and postal code only
- Generated file: `thai-addresses.json`

The storefront loads this static file locally. Customer input is not sent to
GeoThai or any third-party address API.
